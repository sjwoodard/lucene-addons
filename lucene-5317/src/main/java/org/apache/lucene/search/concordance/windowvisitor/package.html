ConcordanceArrayWindowSearcher performs a search on an index and generates a ConcordanceArrayWindow for each hit. This can be very useful for synonym finding and other types of analysis.

To use this searcher, clients can extend ArrayWindowVisitor or use one of the two currently available implementations:

  1. CooccurVisitor -- calculates statistics on terms before and after the target term.
  2. TargetVisitor -- calculates statistics on the target term.
